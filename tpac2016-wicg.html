What's WICG?

Why Incubation?

To enable open, fast, IP-managed iteration on ideas...

... and graceful failure if they don't work out.

Why Incubate in a Single CG?

You don't need to!

... but it's a good firehose-of-ideas community on wicg.discourse.io ...

... and a single CG makes legal signup easier.

What's Cooking?

Lots of stuff! Check it out at https://github.com/WICG/.

All The Incubations!

web-share - Web API proposal for sharing data from a web page IntersectionObserver - API Sketch for Intersection Observers auxclick - Description for non-primary button click event inert - Exploring the `inert` attribute netinfo ResizeObserver interventions - A place for browsers to collaborate on browser interventions. webUSB - Connecting hardware to the web. EventListenerOptions - An extension to the DOM event pattern to allow authors to disable support for preventDefault shape-detection-api - API for hardware assisted detection of shapes, like faces, QR codes, etc. mediasession - Media Session Standard ViewportAPI - A proposal to add explicit APIs to the Web for querying and setting the visual viewport server-timing frame-timing - Frame Timing API media-capabilities ContentPerformancePolicy - A set of policies that a site adhere to, browsers enforce, and embedders can count on. img-conversion - Asynchronously convert between HTMLImageElement, ImageData, and Blob. web-share-target - Web API proposal for receiving shared data media-playback-quality - Media Playback Quality InputDeviceCapabilities - InputDevice API proposal BackgroundSync - A design and spec for ServiceWorker-based background synchronization writable-files - Allowing sites to interact with local files more seamlessly. historical-paymentrequest - Payment Request proposal content-size indexed-db-observers - Prototyping and discussion around indexeddb observers. CSS-Parser-API - This is the repo where the CSS Houdini parser API will be worked on directory-upload - This spec aims to enable directory uploading by allowing a developer to read directory contents (files and sub-directories) asynchronously and be able to identify the directory structure. a11yapi - Accessibility API modality
